summary

  All web developers must know HTML, CSS, and JavaScript to create web pages. In other words, the content of a document can be defined using HTML, how to display the document on the screen can be written using CSS, and JavaScript can be used to create an efficient web page only through a series of collaborations that program the behavior of the web page. Therefore, this course introduces HTML, CSS, and JavaScript, and aims to cultivate the basic ability to create various forms of web pages through their use.
